# **App Name**: Forecap 2025

## Core Features:

- Data Fetching: Fetch story content (images, videos, text) from a single JSON API endpoint.
- Loading State: Implement a loading state that displays while all assets and the API data are being loaded.
- Responsive Story View: Create a story view component that adapts to the width of an iPhone 15 Pro Max while maintaining aspect ratio.
- Automatic Advance: Automatically advance to the next story slide every 10 seconds, with an option to customize the duration.
- Touch Navigation: Enable touch gestures (swipe left/right) for manual navigation between slides.
- Progress Bar: Display a progress bar at the top of the story view, indicating the time remaining on the current slide.

## Style Guidelines:

- Primary color: Deep blue (#293462) to establish a modern and engaging feel.
- Background color: Light gray (#F0F0F0) to ensure high readability and content focus.
- Accent color: Bright pink (#F24A8A) to draw attention to key elements and actions.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and easily readable experience.
- Maximize screen real estate on mobile devices, especially iPhone 15 Pro Max, while maintaining aspect ratio for content.
- Smooth, subtle transitions between slides with a horizontal slide animation. The progress bar should animate in sync with the slide duration.
